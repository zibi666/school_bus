# 学生功能

## 登录

- 学生可以输入**学号**和**密码**登录。

## 包车

- 学生可以**购买车票**来包车。
- 包车的学生的**座位号**固定为**1**。
- 包车时需要学生选定包车**时间**、包车的**车型**、包车的**起始地**、**目的地**。
- 购票后（模拟支付）系统向**管理员**申请**添加车次**，车次的信息为用户所填写的信息。管理员**同意**后系统为该车次随机选取一个司机并添加车次到数据库，其他同学可以输入车次乘坐该车辆。
- 可选**车型**有**客车**和**巴士**，客车核载48人，巴士核载16人，客车包**车价**3888.88元，巴士包**车价**2666.66元。



## 通过输入车次号乘车

​	学生可以填写**车次号**加入到车辆中 ，加入后向**学生购票信息表**添加该学生信息。



## 退票

- 包车的学生（即向管理员发出申请的学生）点击退票后在**学生购票信息表**中删除对应的车次的**所有票**。

- 通过输入车次加入车辆的学生点击退票后直接在**学生购票信息表**中删除其学号对应车票。



## 查看学生购票信息

- 学生可以查看自己的乘车信息，信息内容为学生购票信息表中的内容。
- 学生还可以看到司机的手机号码（从司机表中查询）。

# 管理员功能

## 登录

- 管理员可以输入**账号**和**密码**登录。

## 审核学生的包车申请

- 管理员可以**审核**学生的包车申请。
- 点击同意后系统自动为该申请分配车次号和司机并依据学生申请时填写的日期、出发地等信息向**车次表**添加信息，与此同时向学生购票信息表添加**申请包车的学生**的**购票信息**。



## 查看每个车次的乘车人员信息

- 管理员点击车次号后可以继续**查看该车次的乘车人员**。
- 乘车人员信息从**学生购票信息表**中通过“**WHERE 车次号 = 查询的车次号**”查询。



## 新增司机

- 管理员可以新增司机表中的司机，新增时填写**车牌号**、**车型**等信息。
