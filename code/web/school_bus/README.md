# School Bus Frontend

基于 Vue 3 + Vite + Element Plus 开发的校园包车系统前端页面。

## 🛠 技术栈

- **核心框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI 组件库**: Element Plus
- **路由管理**: Vue Router 4
- **HTTP 客户端**: Axios

## 🚀 快速开始

### 1. 环境准备

- Node.js 16+
- npm 或 yarn

### 2. 安装依赖

在当前目录下运行：

```bash
npm install
```

### 3. 配置环境变量

项目根目录下已创建 `.env` 文件，用于配置后端接口地址：

```properties
VITE_API_BASE_URL=http://localhost:8080/api
```

如果你的后端服务运行在其他地址，请修改此文件。

### 4. 启动开发服务器

```bash
npm run dev
```

启动后访问控制台输出的本地地址（通常是 `http://localhost:5173`）。

## 📱 功能模块

### 学生端
- **注册/登录**: 学号登录，支持填写所在地。
- **包车申请**: 填写目的地、用车时间、车型需求。
- **我的订单**: 查看历史订单状态（审核中/已通过/已拒绝），支持取消审核中的订单。
- **个人中心**: 查看个人基本信息。

### 管理员端
- **登录**: 管理员账号登录。
- **订单审核**: 查看所有申请，进行通过（分配车辆）或拒绝操作。
- **车辆管理**: 录入车辆信息，查看车辆状态（空闲/使用中），删除车辆。
