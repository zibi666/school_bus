# 学生功能

## 购票

- 学生可以购买车票来包车。

- 购票时需要学生选定包车时间、包车的车型、包车的起始地、包车的目的地。

- 购票后（支付模拟一下就行，假的）系统随机选取一个该车型的信息并在车次表中添加信息，生成一个6位邀请码，其他同学可以输入邀请码乘坐购票人包的车辆。

- 可选车型有客车和巴士，客车核载48人，巴士核载22人，客车包车价3888.88元，巴士包车价2999.99元。



## 通过邀请码乘坐车辆

​	学生可以填写购票人分享的邀请码将自己添加到邀请码对应车牌号的乘车人员信息表中并分配座位号，座位号为大于1且小于该车型核载人数的最小的未分配的数字（以防有人退票后空出一个小座位号） 。



## 退票

- 购票包车的学生（即分享邀请码的学生）点击退票后在车次表中删除对应的车次。

- 通过他人分享邀请码乘车的学生点击退票后在乘车人员信息表中删除对应该学生的乘车信息。



## 查看学生购票信息

- 学生可以查看自己的乘车信息，信息内容为学生购票信息表中的内容。
- 学生还可以看到司机的手机号码（从司机表中查询）。

# 管理员功能

## 查看车次

- 管理员可以查看车次表中的所有信息。



## 查看每个车次的乘车人员信息

- 管理员点击车次信息后可以继续查看该车次的乘车人员，乘车人员信息从乘车人员信息表中查询。



## 修改某车次的司机

- 管理员可以修改所选车次的司机，修改后对应车次的所有信息（车次表、学生购票信息表）都需要修改。



## 新增司机

- 管理员可以新增司机表中的司机。
